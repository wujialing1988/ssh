表格面板

扩展类：Ext.grid.Panel

扩展方式：override

扩展功能：

1.移除之前的属性（每个输入框都要有'beforeLabelTextTpl: required'才能出现*）。

2.该输入框只要有'allowBlank'属性，即可出现*。

示例：


示例效果说明：

1.如果输入框是必填项，则只需增加'allowBlank : false'属性即可。

{ fieldLabel : 'IP地址', name : 'roleName', vtype:'IPAddress', allowBlank : false }

用法：

直接引用控件代码即可，不需额外编写代码。

<script type="text/javascript" src="../../scripts/extjs/ux/ExtjsExtend.js"></script>
