单元格提示

扩展类：Ext.grid.column.Column

扩展方式：override

扩展功能：

1.禁用列标题单元格显示按钮功能。

2.将鼠标移动到单元格上时，会出现提示框，显示单元格的内容。

示例：

示例源码：查看

示例后台返回数据：查看

用法：

引入控件JS文件

<script type="text/javascript" src="../../scripts/extjs/ux/ExtjsExtend.js"></script>

扩展功能1，不需额外编写代码。

扩展功能2：

执行EXTJS的 初始化快速提示功能。

Ext.onReady(function() {
	Ext.QuickTips.init();
});

注意：如果在定义列时，设置了renderer属性，则该列不会出现提示效果，需要手动在renderer函数里加上以下代码：

renderer : function(v , ctx , record){ 
	ctx.tdAttr = 'data-qtip="' + v + '"';
	return v; 
}

同理，如果想取消列的消息提示，则需要设置renderer为空函数。即: renderer:function(){}
