查询按钮根据查询条件隐藏相应列

扩展类：Ext.Button

扩展方式：extend

扩展功能：

1.根据每条记录中某个或者多个数据的值去判断列的显示或者隐藏。

2.不需要在查询按钮中加点击事件就能自动查询。

用法：

引入控件JS文件

<script type="text/javascript" src="../../scripts/extjs/ux/ExtjsExtend.js"></script>

参数表：

dataIndex 关联隐藏列的field值，不在查询按钮中配置 需要隐藏的列的dataIndex值，可以是dataIndex:'a&b',表示字段a和字段b的列都隐藏
itemId 关联的查询条件中需要配置是参数，不在查询按钮中配置 通过对查询按钮所在tbar中所有查询条件控件遍历itemId作为参数，控件的值作为参数的值，查询数据，比如itemId : 'keyString',那么keyString作为查询参数

示例：

示例源码：查看

参考代码如下： 

 tbar : ['设备名称', {
		xtype : 'textfield',
		itemId : 'deviceName',
		dataIndex : 'deviceCode&deviceName',
		width : 90
	}, '关键字', {
		xtype : 'textfield',
		itemId : 'keyString',
		width : 90
	}, {
		xtype : 'searchBtn', //查询按钮
	}
]
