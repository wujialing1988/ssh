消息提示

扩展类：Ext.Msg

扩展方式：override

扩展功能：

1.扩展成功消息、警告消息和错误提示消息。

示例：

示例源码：

用法：

引入控件消息JS文件

<script type="text/javascript" src="../../scripts/extjs/ux/TipWindow.js"></script>

调用Ext.Msg对象的方法

Ext.Msg.showTip(SystemConstant.addSuccess); // SystemConstant.addSuccess="添加成功";
Ext.Msg.showWarning('用户session已过期，请重新登录。');
Ext.Msg.showError('网络异常！')

消息常量代码片段

SystemConstant = {
    delSuccess:"删除成功",
    addSuccess:"添加成功",
    editSuccess:"修改成功"
}
